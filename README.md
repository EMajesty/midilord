a midi controller pedal with
- 8 footswitches
- 4x40 character lcd wired in parallel mode
- esp32s3
- 1 midi in, multiple midi outs (wired in parallel from same output or completely separate)
- power from 9v pedal power, regulated down for logic

capabilities
- morningstar-inspired function set
- presets, banks
- deep functionality of a preset/footswitch
- tap tempo
- web server for preset/config editing
